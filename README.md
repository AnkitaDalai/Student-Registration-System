# 🎓 Student Registration System

A responsive and interactive **Student Registration System** built using **HTML, CSS, and JavaScript**.

This project allows users to register student details, edit records, delete records, and store data permanently using **Local Storage**. The application also includes form validation and a modern responsive user interface.

---

# 🚀 Features

* Add new student records
* Edit student details
* Delete student records
* Store data using Local Storage
* Dynamic student table rendering
* Form validation
* Prevent duplicate Student IDs
* Responsive design for mobile and tablet devices
* Modern glassmorphism UI design
* Smooth hover effects and animations

---

# 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage API

# 📋 Functionalities

## ➕ Add Student

Users can register students by entering:

* Student Name
* Student ID
* Email Address
* Contact Number

---

## ✏️ Edit Student

Users can update existing student records using the edit button.

---

## ❌ Delete Student

Users can delete student records after confirmation.

---

## 💾 Local Storage Support

Student data is stored in the browser using Local Storage, allowing records to remain available even after refreshing the page.

---

## ✅ Form Validation

The project includes validation for:

* Empty input fields
* Student name format
* Numeric student IDs
* Valid email addresses
* Contact number length
* Duplicate student IDs
